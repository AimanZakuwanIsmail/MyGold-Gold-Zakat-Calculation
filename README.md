## üìñ About the Project
**MyGold** is a research-backed digital solution developed to simplify zakat calculations for gold owners. This project was funded by a **UiTM University Research Grant** with a specific mission: to digitize the Shariah-compliant calculation process according to the **official rulings of the State of Kedah**.

By working directly with **Lembaga Zakat Negeri Kedah (LZNK)**, the system ensures that every calculation accurately reflects the state's specific *Uruf* (exemption limits) and gold valuation methods.

## ‚ú® Key Features
* **Kedah State Logic:** Built specifically using the official **Lembaga Zakat Negeri Kedah** zakat parameters.
* **Uruf Thresholds:** Automatically applies the specific state-defined weight limits for jewelry and investment gold.
* **Research-Based:** Developed as part of a **UiTM grant-funded** initiative to improve digital Islamic services.
* **Mobile Optimized:** A clean, responsive interface designed for quick access by the Kedah community.
